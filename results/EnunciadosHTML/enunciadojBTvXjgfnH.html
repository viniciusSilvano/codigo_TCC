A orientação a objetos é um paradigma de análise, projeto e programação de sistemas de software baseado na composição e interação entre diversas unidades de software chamadas de objetos. Na programação orientada objetos com Java

  1. as interfaces podem conter assinaturas de métodos, atributos, contantes e construtores.
  2. as classes abstratas podem conter assinaturas de métodos, métodos implementados, constantes, variáveis, construtores e atributos.
  3. tanto as classes abstratas quanto as interfaces podem ser instanciadas diretamente por meio da instrução new.
  4. uma subclasse em uma relação de herança pode herdar muitas superclasses, o que caracteriza herança múltipla.
  5. um método de uma superclasse não pode ser sobrescrito em uma subclasse, porém, pode ser sobrecarregado.
