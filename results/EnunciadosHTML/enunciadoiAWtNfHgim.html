O uso da herança pode otimizar o tempo de desenvolvimento das aplicações orientadas a objetos devido ao reaproveitamento de código. Quando se implementa herança

  1. uma subclasse pode se tornar uma superclasse a qualquer momento, bastando para tanto que se derive uma subclasse a partir dela.
  2. a superclasse herda os atributos da subclasse, incluindo os atributos privados, públicos e protegidos.
  3. múltipla na linguagem Java versão 6, a subclasse herda características de mais de uma superclasse.
  4. só é possível obter polimorfismo na aplicação se houver sobrecarga de métodos.
  5. os métodos de uma superclasse não podem ser sobrescritos nas subclasses. porém, podem ser sobrecarregados.
